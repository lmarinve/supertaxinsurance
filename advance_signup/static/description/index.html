Odoo Advance Signup

Know your customers in the sign up time on your website!

Odoo Advance Sign Up facilitates the Odoo Users to add multiple fields in the sign-up page. Moreover, the user can set some of the fields mandatory for their customers to sign-up.

Registration to any website is an important part so that users can enter the website with the required authentication. Hence, the sign up process needs to be perfect so that you can gather apt information about the customers.

Configuring the sign up page gives you the flexibility to add the fields in the sign up page to collect the information of your visitors. You can then use the information for the better user experience which will help you to build strong relationship clients.

Now, you can create a Custom Odoo Website registration and Signup/login page in the backend with Odoo Advance Signup plugin. The module Odoo Advance Signup adds the functionality to create a desired Odoo website portal signup page and form for your customers. The user can add multiple dynamic fields to the signup form such as mobile number, country, etc.

Configure the Odoo Sign up Page as you want

Now, you can easily configure your Odoo Sign up page in Odoo. Which means that you can easily add as many as fields which need to be filled by the customers.

Moreover, you can also add the background image in the signup page to make it enticing for your customers.

The fields are introduced so that you can gather the information you want from your customers at the time of sign up. It lets you offer the best experience to your visitors with the help of information gathered.

Detailed Features List

Here is the detailed list of features for Odoo Advance Sign Up

  • Odoo Advance Sign Up facilitates the user to design the Sign-Up page for his/her Odoo website
  • The module supports Multi-Website compatible
  • The Odoo admin can add various dynamic fields in the signup form as per the requirements
  • The Odoo admin can set a custom background image for the signup/login/reset password website page
  • The field values are automatically stored in the user data when the customer successfully registers on the Odoo website through the signup form
  • The Admin can also add login/signup/reset password page content from Odoo backend.
  • You can hide/show the header/footer on the Signup page.
  • It is Multi-Language compatible
  • Backend View
  • Website View
  • Change Log
